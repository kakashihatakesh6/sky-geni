# Revenue Intelligence Console

A single-page application for CROs to visualize revenue performance.

## Prerequisites
- Node.js (v18+)

## Setup

1. **Install Backend**:
   ```bash
   cd server
   npm install
   ```

2. **Install Frontend**:
   ```bash
   cd client
   npm install
   ```

## Running the App

1. **Start Backend**:
   ```bash
   cd server
   npm run dev
   ```
   Server will run on `http://localhost:3000`.

2. **Start Frontend**:
   ```bash
   cd client
   npm run dev
   ```
   Open the URL shown (usually `http://localhost:5173`).

## Notes
- "Today" is simulated as **Feb 5, 2026** to match the dataset.
- Data is loaded from `/data` directory.
